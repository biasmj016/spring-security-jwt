# Spring Security JWT with Redis Example

##  주요 기능
- JWT를 사용한 사용자 인증 및 인가
- Redis를 이용한 토큰 저장 및 관리
- 역할 기반 접근 제어(Role-Based Access Control)
- 안전한 비밀번호 암호화(BCrypt 사용)
- 로그아웃 시 토큰 무효화 처리


##  기술 스택
- Java 21
- Gradle
- Spring Boot
    * Spring Security
    * Spring Data Redis
    * Spring Web
- Redis
